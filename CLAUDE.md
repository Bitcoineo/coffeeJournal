# Coffee Journal

## Project
A personal app to log and rate coffee shops I visit.

## Stack
- Vite + React + TypeScript
- Tailwind CSS
- Framer Motion

## Design rules
- Background: #f5f0e8 (warm off-white, like paper)
- Text: #1a1a1a
- Accent: #2a9d8f (teal)
- Secondary accent: #e9c46a (warm yellow)
- Font: Caveat (handwritten headings), DM Sans (body)
- Cards: white, border-radius 16px, subtle box-shadow
- Warm, playful, personal diary energy

## What I'm building
- Log a coffee shop (name, location, rating, notes)
- View all entries
- Filter by rating

## What I'm learning
- Project structure
- CLAUDE.md
- Component architecture
